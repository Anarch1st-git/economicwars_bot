# Telegram Empire Strategy Bot

Telegram-бот с онлайн-стратегией в режиме реального времени. Игрок развивает собственную империю, строит здания, изучает технологии, исследует карту, находит тайники и заброшенные базы, формирует армию, участвует в PvE/PvP и взаимодействует с другими игроками через игровые механики разведки, экономики и сражений.

## Возможности

- Регистрация игрока и создание империи.
- Развитие базы: строительство и улучшение зданий.
- Система исследований и технологий.
- Добыча и управление ресурсами.
- Исследование местности, поиск тайников и опасных локаций.
- PvE-механики: монстры, заброшенные базы, миссии и события.
- PvP-механики: атаки, разведка, шпионаж и взаимодействие с империями других игроков.
- Производство армии и управление боевыми юнитами.
- Сезоны, события, новости и игровые эффекты.
- Магазин, бустеры, предметы и внутриигровой рынок.
- Многоуровневая игровая логика с обработчиками Telegram-команд и callback-кнопок.

## Технологии

- Python
- python-telegram-bot
- PostgreSQL
- Peewee ORM
- Telegram Bot API

## Быстрый запуск

1. Установить зависимости:

```bash
pip install -r requirements.txt
```

2. Создать файл `.env` на основе `.env.example`:

```bash
cp .env.example .env
```

3. Заполнить переменные окружения:

```env
BOT_TOKEN=your_telegram_bot_token

DB_NAME=ew_game_data
DB_USER=postgres
DB_PASSWORD=your_password
DB_HOST=localhost
DB_PORT=5432
```

4. Создать базу данных PostgreSQL.

5. Запустить бота:

```bash
python bot.py
```

## Настройка окружения

Секретные данные не хранятся в коде. Для запуска используются переменные окружения из `.env`:

- `BOT_TOKEN` — токен Telegram-бота.
- `DB_NAME` — имя базы данных.
- `DB_USER` — пользователь PostgreSQL.
- `DB_PASSWORD` — пароль PostgreSQL.
- `DB_HOST` — адрес сервера базы данных.
- `DB_PORT` — порт PostgreSQL.
- `PROVIDER_TOKEN` — платёжный токен, если используются платежи.
- `TEST_PROVIDER_TOKEN` — тестовый платёжный токен.
- `PAYMENT_CURRENCY` — валюта платежей.
